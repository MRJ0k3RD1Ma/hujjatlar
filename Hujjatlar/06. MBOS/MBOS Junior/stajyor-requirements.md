# 🎯 Stajyorlarni Tanlash Uchun Minimum Talablar

## 🔵 Umumiy Talablar (Frontend + Backend uchun)  
- Git va GitHub’dan foydalanishni bilish (repo yaratish, clone, branch, commit, push, pull request).  
- REST API tushunchasiga ega bo‘lish (CRUD operatsiyalarni tushunish).  
- HTTP asoslari (GET, POST, PUT, DELETE, status code).  
- JSON bilan ishlashni bilish.  
- Terminal / CLI asoslari (npm, npx, git buyruqlari).  
- Ingliz tilida dokumentatsiya o‘qiy olish darajasi (kamida texnik atamalarni tushunish).  

---

## 🟢 Frontend (React / Next.js) uchun minimum talablar  
- **HTML5**: Semantik teglar, formalar, input turlari.  
- **CSS3**: Flexbox, Grid, responsive dizayn (media queries).  
- **JavaScript (ES6+)**:  
  - Let/const, arrow functions, template literals  
  - Array methodlari (map, filter, reduce, forEach)  
  - Object destructuring, spread/rest operator  
  - Promise, async/await, fetch API  
- **React.js asoslari**:  
  - Functional components  
  - Props va state  
  - useState, useEffect hook’lari  
  - Component lifecycle tushunchasi  
  - Form handling va controlled components  
- **Next.js asoslari (bonus bo‘lsa yaxshi)**:  
  - Pages va routing  
  - getStaticProps / getServerSideProps tushunchasi  
- **UI Library**: TailwindCSS yoki Bootstrap asosiy darajada bilish.  

---

## 🟠 Backend (Node.js / Express / NestJS) uchun minimum talablar  
- **JavaScript (ES6+)**:  
  - Variables, functions, scope  
  - Array va object metodlari  
  - Async/await, promise chaining  
- **Node.js asoslari**:  
  - npm package manager’dan foydalanish  
  - Express.js’da oddiy CRUD API yozish  
  - Middleware nima ekanini tushunish  
- **REST API**:  
  - Route’lar, params, query string, body  
  - Status code’lar (200, 201, 400, 404, 500)  
- **Postgres yoki SQL asoslari**:  
  - CREATE TABLE, SELECT, INSERT, UPDATE, DELETE  
  - Simple relationships (one-to-many) tushunchasi  
- **NestJS (bonus, majburiy emas)**:  
  - Module, Controller, Service arxitekturasi haqida umumiy tushuncha.  

---

## 🟣 Soft Skills (majburiy emas, lekin ustunlik beradi)  
- Jamoada ishlash tajribasi bo‘lsa (hatto kichik loyihalarda ham).  
- Trello / Jira yoki boshqa task manager ishlatgan bo‘lsa.  
- Kodni toza yozishga e’tibor (linting, naming convention).  
- Problem-solving mindset (googling skills 🔍).  

---

## 📝 Tanlov jarayoni (Tavsiya etiladi)  
- **Test (30 daqiqa)**: JS asosiy savollar + SQL oddiy query + HTML/CSS qisqa topshiriq.  
- **Amaliy topshiriq (1-2 soat)**: Oddiy CRUD API yoki React’da Todo app.  
- **Suhbat**: Jamoada ishlash qobiliyati va motivatsiyani baholash.
